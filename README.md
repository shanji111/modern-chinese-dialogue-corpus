# 汉语对话语料库 Chinese Dialogue Corpus

> A full-stack corpus platform for Chinese dialogic data, designed for research in dialogic syntax, interactional linguistics, corpus linguistics, and AI-assisted dialogue analysis.

**在线访问：** https://chinese-dialogue-corpus.onrender.com/

---

## 项目简介

**汉语对话语料库**是一个围绕真实汉语互动材料建设的语料检索与研究平台。项目以“对话转向”（Dialogic Turn）为基本视角，不将语料仅视为静态文本集合，而是尽量保留话轮结构、上下文关系、说话人信息、相邻回应和多模态线索，使汉语对话材料能够被检索、回溯、比较和进一步分析。

本项目服务于语料库语言学、互动语言学、语用学、话语分析、对话句法和认知语言学等研究方向，也面向生成式人工智能时代的对话建模、语言理解评测与语言知识组织需求。平台尝试将真实互动材料从“可保存的语言资源”推进为“可分析的互动知识基础设施”。

## 项目定位

本项目的核心目标不是单纯收集中文句子，而是建设一个以自然互动为核心的汉语对话研究平台。项目重点关注：

- 汉语真实互动材料的收集、清洗、结构化整理与检索；
- 话轮结构、相邻回应、说话人关系和上下文链条的保留；
- 对话中的重复、承接、否定、修复、问答、立场表达等互动现象；
- 对话句法理论中的结构复现、语义映射与共鸣关系；
- 生成式人工智能时代对话材料的组织、评测与语言知识建模。

平台希望将汉语对话材料从静态文本资源推进为具备结构化、可检索性和可解释性的互动知识空间。

## 核心功能

### 1. 多类型语料检索与浏览

平台前台围绕多种汉语对话材料组织语料，包括：

- 日常对话
- 影视对白
- 古代汉语
- 文本对话
- 网络回帖
- 访谈语料
- 课堂互动
- 多模态语料

检索结果保留来源、类别、数据集、年份、标题、说话人、对话编号等元信息，便于进行样例回溯和语境分析。

### 2. 多种检索方式

平台目前支持多种互补的检索入口：

- **普通检索**：用于检索固定词语、短语和高频表达；
- **正则检索**：用于检索形式变体、边界限制和间隔模式；
- **高级检索**：支持来源、标题、年份、说话人、正文长度、是否带音频等字段限制；
- **对话句法 / 共鸣检索**：基于相邻话轮关系，观察重复、承接、否定、问答、修复等互动模式。

### 3. 上下文保留与对话回溯

检索结果并非只呈现孤立命中句，而是尽量保留互动语境。平台目前提供：

- **居中检索视图**：以前序话轮、命中话轮、后续话轮展示局部上下文；
- **完整对话视图**：展示命中所在的完整对话或较大对话片段；
- **元数据展示**：保留来源、作品、章节、说话人、对话 ID、音频入口等信息。

这一设计使语料检索能够服务于真实语境中的语言分析，而不仅是词语层面的频次查询。

### 4. 投稿、审核与转写

平台内置轻量级投稿与后台审核流程，支持后续语料扩展：

- 支持文本、TXT、音频、视频等材料提交；
- 支持后台审核、下载、删除和批准投稿；
- 支持音频 / 视频材料的自动转写；
- 审核通过的材料可写入主库或多模态语料表。

### 5. 文本对话整理流水线

仓库中的 `talkdata/` 目录用于文本对话子库的工程化整理。相关脚本主要服务于：

- 标准化 CSV 导入；
- 本地 / 线上语料状态检查；
- 滑动窗口式对话单元合并；
- 访谈样本生成；
- 专项语料导入；
- LLM 辅助审核结果整理。

大量原始语料、LLM 缓存、审核导出文件和中间结果默认不进入公开仓库，以保证仓库结构清晰和数据边界可控。

## 数据结构概览

本项目的数据结构围绕“对话”而非单句文本设计。除主表外，平台还构建了全文检索表和对话结构派生表。

### 主表

- `corpus_entries`  
  保存标题、正文、来源、年份、类别、数据集名、说话人、对话片段等主语料信息。

### 检索派生表

- `corpus_entries_fts`  
  SQLite FTS5 全文索引表，用于普通检索、高级检索和部分正则检索流程。

### 对话结构派生表

- `dialogue_turns`  
  从主表内容中拆分出的**话轮级数据**，服务于相邻话轮分析。

- `dialogue_pairs`  
  从 `dialogue_turns` 派生出的**相邻话轮对数据**，服务于对话句法、共鸣检索和互动模式分析。

### 投稿与多模态相关表

- `corpus_submissions`  
  保存用户投稿、审核状态与后台处理信息。

- `multimodal_entries`  
  保存与音频、视频和多模态材料相关的条目信息。

## 公开范围说明

本仓库主要用于展示项目结构、代码框架、功能设计与语料库平台建设思路，仅作为学术交流、技术展示与研究方法说明使用。

出于研究伦理、版权边界、数据安全和课题管理要求，仓库不提供以下内容：

- 完整原始语料；
- 未经整理或授权的中间数据；
- 教师课题组尚未公开的研究材料；
- 用户投稿内容；
- 音频、视频及其原始转写文件；
- 数据库完整备份文件；
- 涉及个人信息、版权文本或平台来源限制的材料。

公开仓库仅保留必要的代码、文档、少量示例数据和项目说明，用于展示平台功能与技术实现路径。完整语料与研究数据由课题组或研究者在本地及合规存储环境中管理，不随本仓库公开发布。

## 项目意义

汉语对话语料库不仅是一个在线检索网站，也是一项面向语言学研究的数字基础设施实践。它将多来源汉语对话材料整理为可检索、可回溯、可比较、可扩展的互动知识空间，为对话句法、互动语言学、语料库语言学和 AI 辅助语言研究提供基础数据与工具支持。
